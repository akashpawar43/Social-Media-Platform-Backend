### create .env file and add below code 

```
PORT = 4000
JWT_SECRET = "add_your_jwt_secret"
MONGO_URL = "add_your_mongo_url"
```

### Install dependencies

```
npm install
```

### Start Server

```
npm start
```

### viste below url for api documentation

